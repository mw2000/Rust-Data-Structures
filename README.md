# Introduction
This is a project to recreate the popular ADT's in Rust

## ADT's Implemented:
- Fixed List
- Stack
- Queue
- Deque
- Steque
- Hash Table
- Graph

## ADT's In-Progress:
- Linked List
- Binary Heap

## ADT's To-Be-Made:
- Doubly Linked List
- Dynamic List
- BST
- AVL Trees
- Red Black Trees
- Left Leaning RB Trees
- Splay Trees
- B Trees
- Trie
- Binary Heap
- Priority Queue
- Indexed Priority Queue
